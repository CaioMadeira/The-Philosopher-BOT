Hexadecimal 0123456789ABCDEF

--- Esse é o dicionário que usamos pra marcar as cores.
	Conseguimos enxergar o espectro RGB que quer dizer
	RED, GREEN and BLUE. 

	Montamos as cores seguindo essas mesmas caracteristicas.
	Usando uma tralha com 6 elementos.

	# _ _ _ _ _ _ 

	-> Os dois primeiros para o vermelho
	-> Os dois segundos para o verde
	-> Os dois últimos para marcar o azul

	Em hexadecimal:

	0 = ausência de cor

	F = máximo de cor

	#00 00 00 -> cada cor (VERMELHO, VERDE E AZUL) recebem 0 de cor,
	ou seja, total ausência.

	#FF FF FF -> Aqui todas recebem o máximo possível de cor para
	todos os pares de cores.

	VERMELHO -> #FF0000

	VERDE -> #00FF00

	AZUL -> #0000FF


RGB - Red, Green, Blue 

	--- O alfabeto de coresque vai do 0 até o 255.

	0 = ausência de cor

	255 = máximo de cor
